GET Endpoints

GET Get All Categories

GET /api/shop/all

Returns a list of all categories with their subcategories

Response Example:

{
    "id": 1,
    "name": "Áo nam",
    "subcategories": [
        {
            "id": 1,
            "name": "Áo Polo",
            "img": "../img/shared/subcategory/ao_polo.png"
        }
    ]
}

GET Get All Items

GET /api/items

Returns a list of all items

Response Example:

{
    "id": 1,
    "name": "Áo Polo Nam Pique Mắt Chim Basic",
    "sold": 621,
    "price": 299000,
    "discount": 10,
    "colors": ["Xanh", "Đen", "Trắng"],
    "sizes": ["M", "L", "XL"]
}

GET Get Items By Category

GET /api/items/category/{categoryId}

Returns items for a specific category

GET Get Item by ID

GET /api/items/{id}

Returns a specific item by its ID

Parameters:

Parameter Type Required Description
id number Yes Item ID to retrieve

Response Example:

{
    "id": 1,
    "name": "Áo Polo Nam Pique Mắt Chim Basic",
    "sold": 621,
    "price": 299000,
    "discount": 10,
    "subcategoryId": 1,
    "colors": ["Xanh", "Đen", "Trắng"],
    "sizes": ["M", "L", "XL"],
    "characteristics": [
        "Vải mắt chim thông thoáng",
        "Thấm hút tốt"
    ]
}

POST Endpoints

POST Create New Item

POST /api/items

Creates a new item

Request Body:

{
    "name": "Áo Polo Nam Mới",
    "price": 299000,
    "discount": 10,
    "colors": ["Xanh", "Đen"],
    "sizes": ["M", "L", "XL"],
    "subcategoryId": 1
}

PUT Endpoints

PUT Update Item

PUT /api/items/{id}

Updates an existing item

Parameters:

Parameter Type Required Description
id number Yes Item ID to update

Request Body:

{
    "name": "Áo Polo Nam Updated",
    "price": 399000,
    "discount": 15
}

DELETE Endpoints

DELETE Delete Item

DELETE /api/items/{id}

Deletes an item

Parameters:

Parameter Type Required Description
id number Yes Item ID to delete

Search API

GET Search Items

GET /api/items/search?query={searchTerm}

Search items by name, color, size, or price range

Query Parameters:

Parameter Type Required Description
query string Yes Search term to filter items
