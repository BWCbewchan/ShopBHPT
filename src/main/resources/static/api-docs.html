BHPT API Documentation

RESTful API documentation for BHPT E-commerce Platform

Categories

GET Get All Categories

GET /api/shop/all

Returns a list of all categories with their subcategories

Example Response:

{
    "id": 1,
    "name": "Áo nam",
    "subcategories": [
        {
            "id": 1,
            "name": "Áo Polo",
            "img": "../img/shared/subcategory/ao_polo.png"
        }
    ]
}

Items

GET Get All Items

GET /api/items

Returns a list of all items

Example Response:

{
    "id": 1,
    "name": "Áo Polo Nam Pique Mắt Chim Basic",
    "sold": 621,
    "price": 299000,
    "discount": 10,
    "colors": ["Xanh", "Đen", "Trắng"],
    "sizes": ["M", "L", "XL"],
    "characteristics": [
        "Chất liệu: Pique mắt chim",
        "Cotton USA cao cấp",
        "Thấm hút tốt"
    ]
}

GET Get Item by ID

GET /api/items/{id}

Returns a single item by its ID

Parameters:

id: number (path parameter)

POST Create Item

POST /api/items

Creates a new item

Request Body:

{
    "name": "Áo Polo Nam Pique Mắt Chim Basic",
    "price": 299000,
    "discount": 10,
    "colors": ["Xanh", "Đen", "Trắng"],
    "sizes": ["M", "L", "XL"],
    "characteristics": [
        "Chất liệu: Pique mắt chim",
        "Cotton USA cao cấp",
        "Thấm hút tốt"
    ],
    "subcategoryId": 1
}

PUT Update Item

PUT /api/items/{id}

Updates an existing item

Parameters:

id: number (path parameter)

Request Body:

{
    "name": "Áo Polo Nam Updated",
    "price": 399000,
    "discount": 15
}

DELETE Delete Item

DELETE /api/items/{id}

Deletes an item

Parameters:

id: number (path parameter)

Subcategories

GET Get All Subcategories

GET /api/subcategories

Returns a list of all subcategories

Example Response:

{
    "id": 1,
    "name": "Áo Polo",
    "img": "../img/shared/subcategory/ao_polo.png",
    "categoryId": 1
}
